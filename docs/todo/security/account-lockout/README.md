# Account Lockout After Failed Attempts

## Status

**Not Started** | In Progress | Complete

## Summary

Implement account lockout functionality to protect user accounts from brute-force attacks by temporarily locking accounts after multiple failed login attempts.

## Priority

**Medium** (from roadmap)

## Requirements

- [ ] Lock account after 5 failed login attempts
- [ ] Configurable lockout duration
- [ ] Unlock via email
- [ ] Admin unlock capability
