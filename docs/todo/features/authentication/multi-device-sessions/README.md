# Multiple Device Session Handling

## Status

**Not Started** | In Progress | Complete

## Summary

Enables users to manage their sessions across multiple devices. Users can view all active sessions, remotely terminate sessions on other devices, and receive notifications when their account is accessed from a new device.

## Priority

**Low** (from roadmap)

## Requirements

- [ ] View active sessions
- [ ] Remote session termination
- [ ] New device notifications
