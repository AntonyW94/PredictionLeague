# Features

This folder contains feature plans organised by category.

## Categories

| Category | Description |
|----------|-------------|
| [legal-compliance](legal-compliance/) | Privacy, terms, GDPR, cookies |
| [authentication](authentication/) | Login, password reset, sessions, 2FA |
| [email-notifications](email-notifications/) | Transactional emails, reminders, preferences |
| [user-experience](user-experience/) | UI/UX, accessibility, mobile, gamification |
| [admin-moderation](admin-moderation/) | Admin tools, user management, moderation |

## Priority Guide

- **Critical** - Required for public launch
- **Medium** - Important for good user experience
- **Low** - Post-launch improvements and roadmap items
